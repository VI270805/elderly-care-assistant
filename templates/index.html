Frequently Asked Questions
Will my reminders be saved if I close the app?
Yes — all reminders and health records are stored on the Flask server per session. For permanent persistence across server restarts, the in-memory store can be swapped for a database like SQLite or PostgreSQL.
How does the SOS button work?
Save an emergency contact first (name + phone). Then press the big red SOS button — it simulates calling your contact. In a production version, this would trigger a real call via Twilio or send an SMS alert.
Does the Voice Assistant need internet?
The Web Speech API (Chrome / Edge / Safari) uses Google's cloud servers, so an internet connection is needed. On unsupported browsers, a smooth typing-simulation fallback kicks in automatically.
How do I set up the AI chatbot?
Set your OpenAI API key before starting the server: export OPENAI_API_KEY=sk-... then run python app.py. All API calls are made server-side so your key is never exposed to the browser.
What do the health status colours mean?
● Green = Normal  ·  ● Blue = Low  ·  ● Amber = Elevated  ·  ● Red = High

Ranges follow standard clinical guidelines (e.g. BP normal ≤120/80, sugar normal 70–140 mg/dL).
Can multiple users log in?
Yes — the Flask backend stores data per username. Add more users to the USERS dictionary in app.py. Each user gets completely separate medicine and health records.
